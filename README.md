Идея:
  1. Сервис для музыкантов. Поиск постоянных или сессионных музыкантов.
  У каждого участника есть анкета, в который указаны инструменты и жанры, участие в других проектах, демки и тд. Поиск постоянных организован через анкеты. Поиск сессионных имеет вид своеобразной биржи: есть внутренняя валюта, заработать её можно записывая парты для других участников системы, тратить - покупая чужие парты.



Варианты названия:
  Bandmate


Экраны:
  1.Splash Screen
  2.Авторизация 
  3.Регистрация 
  4.Профиль 
    В профиле можно выбрать свои инструменты и жанры, сменить аватарку, посмотреть баланс
  5.Поиск (свайпы) 
    Есть фильтры по инструментам, жанрам, городу, полу, возрасту.
  6.Поиск (анкеты) 
  7.Чат
  8.Биржа 
    На бирже есть внутренняя валюта. Можно создавать и откликаться на объявления, отдавая или приобретая внутреннюю валюту соответственно
